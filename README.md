# Vaa Tamil Siddha

Static website for Vaa Tamil Siddha.

## Folder Structure

```text
vaa_tamil_siddha/
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── images/
│       └── site images and icons
└── docs/
    └── previews/
        └── design and browser preview screenshots
```

## Notes

- Keep production images in `assets/images/`.
- Keep generated screenshots and visual QA previews in `docs/previews/`.
- Open `index.html` directly in a browser to view the site.
